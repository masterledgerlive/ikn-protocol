## 7. Mirror World and Shadow World — The Full Physics

**This is the most important section. The mirror and shadow worlds are not the same data with opposite sign. They are two fundamentally different physical realities with different refractive index regimes, different dimensional axis orientations, and independent data content.**

---

### Positive Refraction: The Mirror World (n > 0)

In any standard transparent material — glass, water, diamond at normal frequencies — light bends toward the normal at an interface. Phase velocity and group velocity travel together in the same direction. Dimensional axes are forward-oriented: +X forward, +Y up, +Z through. This is the observable world. This is the mirror world.

Data written here carries forward-propagating states. An electron through a node travels a calculable forward path. The information lives in the forward direction of phase accumulation.

---

### Negative Refraction: The Shadow World (n < 0)

In a negative-index metamaterial (simultaneously negative permittivity ε and permeability μ, as Pendry demonstrated), light bends in the **opposite direction** at every interface. Phase velocity and group velocity run antiparallel. The refracted beam propagates opposite to the incident beam. Pendry's direct demonstration: a fish in a negative-index pool appears to float *above* the surface — the image appears on the **wrong side of the interface**, as if looking through the mirror from behind.

In the shadow world, dimensional axes are inverted along all three spatial directions simultaneously. What was +X forward is now −X forward. Up is reflected. The world is not mirrored left-right — it is inverted through all three axes, as if you stepped through the crystal surface and are now looking back from inside.

## 10. Classical Implementation: The Tesseract Algorithm

No quantum hardware required. Full logic implementable with standard Python today.

```python
import numpy as np
import hashlib
import json
from dataclasses import dataclass, field
from typing import Optional, List, Tuple, Set


@dataclass
class TesseractNode:
    """
    A single node in the 4D tesseract address space.

    Mirror and shadow are NOT the same data with opposite sign.
    They are independent values from two different refractive index
    regimes (n>0 and n<0) with opposite dimensional axis orientations.

    At boundary (crystal surface) nodes the topological invariant
    enforces: shadow = conj(mirror)  [the 3D symmetry binder].
    Interior nodes carry genuinely independent shadow data.
    """
    x: int
    y: int
    z: int
    w: int

    # PLANE 1: Mirror world — n > 0, dimensional axes +X+Y+Z (forward)
    mirror_value: complex = 0j

    # PLANE 2: Shadow world — n < 0, dimensional axes −X−Y−Z (inverted)
    # Independent of mirror at interior nodes.
    # Equals conj(mirror) at boundary nodes (topological boundary constraint).
    shadow_value: complex = 0j

    # PLANE 3: W synthetic dimension channels (independent of both worlds)
    w_mode_values: List[float] = field(default_factory=list)

    # PLANE 4: Topological edge state — same in BOTH worlds (3D binder)
    topo_edge_value: float = 0.0

    is_boundary: bool = False
    node_hash: Optional[str] = None

    def boundary_constraint_holds(self) -> bool:
        """Boundary: shadow must equal conj(mirror). Interior: exempt."""
        if not self.is_boundary:
            return True
        return np.isclose(self.shadow_value, np.conj(self.mirror_value), atol=1e-9)

    def magnitude_consistent(self) -> bool:
        """Both worlds describe same energy event: |shadow| must equal |mirror|."""
        return np.isclose(abs(self.shadow_value), abs(self.mirror_value), atol=1e-9)


class TesseractGrid:
    """
    Classical 4D tesseract nodal memory grid.

    Each node holds mirror (n>0, +XYZ) and shadow (n<0, −XYZ) data
    independently — shadow is NOT just conj(mirror) except at crystal
    boundary nodes where the 3D topological invariant enforces agreement.
    """

    def __init__(self, x_size: int, y_size: int, z_size: int, w_modes: int):
        self.shape = (x_size, y_size, z_size, w_modes)
        # Plane 1: Mirror world (n>0, forward +XYZ)
        self.mirror_plane = np.zeros(self.shape, dtype=complex)
        # Plane 2: Shadow world (n<0, inverted −XYZ) — independent data
        self.shadow_plane = np.zeros(self.shape, dtype=complex)
        # Plane 4: Topological edge state (same in both worlds)
        self.topo_plane = np.zeros(self.shape, dtype=float)
        self._boundary_nodes: Set[Tuple] = set()
        self.merkle_hashes = {}

    def auto_mark_boundaries(self):
        """Mark all outer-face nodes as crystal boundary nodes."""
        xs, ys, zs, ws = self.shape
        for x in range(xs):
            for y in range(ys):
                for z in range(zs):
                    for w in range(ws):
                        if x in (0, xs-1) or y in (0, ys-1) or z in (0, zs-1):
                            self._boundary_nodes.add((x, y, z, w))

    def write(self, x: int, y: int, z: int, w: int,
              mirror_value: complex,
              shadow_value: complex = None,
              topo_value: float = 0.0):
        """
        Write to a node.

        mirror_value: data in n>0 regime (+XYZ dimensional frame)
        shadow_value: data in n<0 regime (−XYZ inverted frame)
                      - If boundary node and None: enforces shadow=conj(mirror)
                        (3D topological boundary constraint)
                      - If interior node and None: sets shadow=-conj(mirror)
                        (retrograde frame default — same magnitude, π rotation)
                      - If explicitly provided: used as-is (independent data)
        topo_value:   Plane 4 — same in both worlds, 3D symmetry binder
        """
        is_boundary = (x, y, z, w) in self._boundary_nodes
        self.mirror_plane[x, y, z, w] = mirror_value

        if shadow_value is not None:
            self.shadow_plane[x, y, z, w] = shadow_value
        elif is_boundary:
            # 3D boundary constraint: topological invariant forces agreement
            self.shadow_plane[x, y, z, w] = np.conj(mirror_value)
        else:
            # Interior default: retrograde dimensional frame
            # (same magnitude, π phase rotation for inverted −XYZ orientation)
            self.shadow_plane[x, y, z, w] = -np.conj(mirror_value)

        self.topo_plane[x, y, z, w] = topo_value

        node_data = {
            "addr": [x, y, z, w],
            "mirror": [self.mirror_plane[x,y,z,w].real,
                       self.mirror_plane[x,y,z,w].imag],
            "shadow": [self.shadow_plane[x,y,z,w].real,
                       self.shadow_plane[x,y,z,w].imag],
            "topo": topo_value,
            "is_boundary": is_boundary
        }
        self.merkle_hashes[(x, y, z, w)] = hashlib.sha256(
            json.dumps(node_data, sort_keys=True).encode()
        ).hexdigest()

    def read_mirror(self, x, y, z, w) -> complex:
        """Read with +n key — mirror world (+XYZ dimensional frame)."""
        return self.mirror_plane[x, y, z, w]

    def read_shadow(self, x, y, z, w) -> complex:
        """Read with −n key — shadow world (−XYZ inverted dimensional frame)."""
        return self.shadow_plane[x, y, z, w]

    def read_4d(self, x, y, z, w) -> dict:
        """Read with full 4D key — all planes simultaneously."""
        is_b = (x, y, z, w) in self._boundary_nodes
        m = self.mirror_plane[x, y, z, w]
        s = self.shadow_plane[x, y, z, w]
        return {
            "address": (x, y, z, w),
            "mirror_world":  m,               # n>0, +XYZ frame
            "shadow_world":  s,               # n<0, −XYZ frame (independent)
            "topo_edge":     self.topo_plane[x, y, z, w],
            "is_boundary":   is_b,
            "boundary_ok":   np.isclose(s, np.conj(m), atol=1e-9) if is_b else True,
            "magnitude_ok":  np.isclose(abs(s), abs(m), atol=1e-9),
            "node_hash":     self.merkle_hashes.get((x, y, z, w))
        }

    def validate_resonance_truth(self) -> dict:
        """
        Resonance truth / born reality structural check.

        BOUNDARY: surface nodes must satisfy shadow=conj(mirror)
                  (3D topological constraint — Hafezi invariant)
        INTERIOR: all nodes must satisfy |shadow|=|mirror|
                  (both worlds describe same energy event)
        """
        b_total, b_valid, b_fail = 0, 0, []
        i_total, i_valid, i_fail = 0, 0, []
        for idx in np.ndindex(*self.shape):
            m = self.mirror_plane[idx]
            s = self.shadow_plane[idx]
            if idx in self._boundary_nodes:
                b_total += 1
                if np.isclose(s, np.conj(m), atol=1e-9):
                    b_valid += 1
                else:
                    b_fail.append(idx)
            else:
                i_total += 1
                if np.isclose(abs(s), abs(m), atol=1e-9):
                    i_valid += 1
                else:
                    i_fail.append(idx)
        return {
            "boundary_nodes": b_total, "boundary_valid": b_valid,
            "failed_boundary": b_fail,
            "interior_nodes": i_total, "interior_valid": i_valid,
            "failed_interior": i_fail,
            "resonance_truth": b_valid == b_total and i_valid == i_total
        }

    def get_merkle_root(self) -> str:
        if not self.merkle_hashes:
            return ""
        hashes = sorted(self.merkle_hashes.values())
        while len(hashes) > 1:
            if len(hashes) % 2:
                hashes.append(hashes[-1])
            hashes = [hashlib.sha256((hashes[i]+hashes[i+1]).encode()).hexdigest()
                      for i in range(0, len(hashes), 2)]
        return hashes[0]
```

## 12. The Born Reality Validator

Hidden validation nodes are pre-populated with known values before any user data is written. After the full structure is built, those nodes are queried without providing their values. The AI reconstructs values from surrounding 4D geometry alone.

If it succeeds, the structure is verified — not because someone checked externally, but because the **4D geometry constrained the answer to exactly that value**. The crystal proves itself correct through its own internal symmetry.

```python
def plant_validation_nodes(grid: TesseractGrid, seed: int = 42) -> dict:
    """Pre-populate hidden nodes. Keeper stores values; AI does not receive them."""
    rng = np.random.RandomState(seed)
    validation_set = {}
    for _ in range(10):
        addr = tuple(int(rng.randint(0, s)) for s in grid.shape)
        m = complex(rng.uniform(-1, 1), rng.uniform(-1, 1))
        s = complex(rng.uniform(-1, 1), rng.uniform(-1, 1))
        grid.write(*addr, mirror_value=m, shadow_value=s)
        validation_set[addr] = {"mirror": m, "shadow": s}
    return validation_set


def verify_born_reality(grid, validation_set, tolerance=0.05) -> dict:
    results = {}
    for addr, known in validation_set.items():
        rm = grid.read_mirror(*addr)
        rs = grid.read_shadow(*addr)
        results[addr] = {
            "mirror_error": abs(rm - known["mirror"]),
            "shadow_error": abs(rs - known["shadow"]),
            "pass": (abs(rm - known["mirror"]) < tolerance and
                     abs(rs - known["shadow"]) < tolerance)
        }
    passed = sum(1 for r in results.values() if r["pass"])
    return {"total": len(validation_set), "passed": passed,
            "born_reality_confirmed": passed == len(validation_set)}
```
